Financial API Playground

Use this page to call every endpoint in your Flask app directly from the browser. It sends the same requests your API clients would send and shows the raw JSON response.

JSON Overview Sample Company Sample Market
GET /api/company/<symbol>

Company Profile

Fetch summary, sector, website, employees, and key officers for a stock symbol.

GET /api/market/<symbol>

Market Snapshot

Fetch the latest trading price, change, trading range, and volume information.

POST /api/historical

Historical Prices

Fetch daily OHLCV records for a symbol across a date range.

POST /api/insights

Insights

Compute return, volatility, moving averages, trend, and plain-English insights.
